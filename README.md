# Analisis Penyakit ISPA - Frontend UI

Frontend UI for the ISPA Expert System, built with Next.js, Tailwind CSS, and Zustand.

## Prerequisites
- Node.js (v18+)
- npm

## Cara Menjalankan (How to Run)

1. Buka terminal dan masuk ke folder frontend:
   ```bash
   cd analisis-penyakit-ispa-frontend
   ```

2. Install dependencies (jika belum pernah di-install sebelumnya):
   ```bash
   npm install
   ```

3. Jalankan Development Server:
   ```bash
   npm run dev
   ```

4. Buka browser dan kunjungi `http://localhost:3000`.

## Catatan Tambahan
- Pastikan backend server Django (`http://localhost:8000`) sudah berjalan secara bersamaan agar frontend dapat terhubung ke API untuk melakukan login, registrasi, dan konsultasi.
